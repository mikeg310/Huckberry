# Huckberry x Gladly

A personalized sales page comparing Huckberry's current Zendesk + Alhina CX stack against Gladly's platform.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/nlarkin1986/huckberry-gladly-page)

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
